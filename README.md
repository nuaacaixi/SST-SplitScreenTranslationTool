# Translation Split View

一个 Chrome 扩展，在任意网页上提供**分屏翻译视图**——一侧显示 Google 翻译版本，另一侧显示原文，两侧滚动同步。适合阅读英文文章时对照原文使用。

## 功能

- **一键切换** —— 点击工具栏图标即可在当前页面开启/关闭分屏视图
- **Google 翻译集成** —— 自动检测源语言，翻译成你选择的目标语言
- **滚动同步** —— 两侧面板的滚动位置实时同步（基于百分比，带防抖和防回弹机制）
- **可拖拽分割条** —— 在 30%~70% 之间自由调整分屏比例
- **布局切换** —— 支持左右（水平）和上下（垂直）两种布局
- **面板交换** —— 一键将翻译面板和原文面板互换位置
- **偏好设置** —— 目标语言、默认布局、翻译位置、默认分割比例均可配置，通过 Chrome 同步跨设备保存
- **Esc 关闭** —— 按 `Esc` 键快速退出分屏

## 安装

1. 克隆或下载本项目到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目根目录

## 使用

1. 打开任意英文（或其他语言）网页
2. 点击浏览器工具栏中的扩展图标
3. 页面会被分为两个面板，一侧显示 Google 翻译结果，另一侧显示原文
4. 拖动中间的分割条调整比例；点击控制栏按钮切换布局或交换面板
5. 再次点击图标或按 `Esc` 退出分屏

## 设置

点击扩展图标右键 →「选项」，或在 `chrome://extensions/` 中找到该扩展的详细信息 →「扩展程序选项」。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 目标语言 | 翻译的目标语言（支持中/日/韩/英/法/德/西/葡/俄/阿） | 简体中文 |
| 默认布局 | 水平（左右）或垂直（上下） | 水平 |
| 翻译位置 | 翻译面板放在左侧/上侧或右侧/下侧 | 左侧 |
| 默认分割比例 | 两个面板的初始占比 | 50% |

## 技术架构

- **Manifest V3** Service Worker + Content Script
- 纯原生 JavaScript，无任何依赖
- 通过 `postMessage` 实现跨 iframe 滚动同步
- `chrome.storage.sync` 持久化用户偏好

```
工具栏图标点击 → Service Worker → Content Script (顶层)
                                      ├── iframe: Google 翻译页面
                                      └── iframe: 原始页面
                                            ↕ postMessage 滚动同步
```
